# Rock Paper Scissors Game

A classic Rock Paper Scissors game implementation using HTML, CSS, and JavaScript.

## Project Structure

```
├── index.html
├── style.css
├── script.js
├── Images/
├── LICENSE
└── README.md
```

## Features

- Interactive game interface
- Player vs Computer gameplay
- Score tracking
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Start playing!

## How to Play

1. Click on your choice (Rock, Paper, or Scissors)
2. The computer will make its choice
3. Winner will be determined according to classic rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
